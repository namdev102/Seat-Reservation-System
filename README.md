# Seat-Reservation-System
The interface has the following capabilities:

▪ Interface has an input form for the user to enter their name and the number of seats they

want to book.

▪ The seat map is clickable to select seat. The user clicks on empty seats to select

them. Clicking on a selected seat will deselect it. The user is allowed to select

only as many seats as they have entered.

▪ The page also contains a table that shows the list of users that have reserved seats,

the number of seats each has selected and the seat nos. of the seats they have reserved.
